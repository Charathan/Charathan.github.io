Unity

public class ChapterSwitcher : MonoBehaviour {

   private static ChapterSwitcher instance;

   public static ChapterSwitcher Instance

   {

       get { return instance; }

   }

private void Awake()

{

      if (instance != null && instance != this)

      {

          Destroy(gameObject);

          return;

      }

      instance = this;

      DontDestroyOnLoad(gameObject);

}

}

Chapterswitcher

Een van de grote scripts waar ik aan gewerkt heb voor zorgt voor het switchen van scenes. De class gebruikt een message queue om berichten op te slaan. Op basis van deze berichten kunnen dingen in de chapter worden veranderd. De berichten kunnen aan de chapter switcher worden gegeven door middel van een functie. Het object kijk dan elke frame of er berichten zijn en voert deze uit.
